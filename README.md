Simple utility for checking JPEG images for integrity after downloading

Usage: 

main.py [-h] -p PATH [-r]

options:\
&emsp;-h, --help\
&emsp;&emsp;show this help message and exit\
&emsp;-r, --recursively\
&emsp;&emsp;including subdirectories

required arguments:\
&emsp;-p PATH, --path PATH\
&emsp;&emsp;path to check
